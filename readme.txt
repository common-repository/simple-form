== Frequently Asked Questions ===

= How to Create a WhatsApp Form in WordPress? =
It's quite easy to get started with the FormFlow plugin. To get started, activate the FormFlow(simple form), create a form using drag and drop builder, then visit WhatsApp Tabs- FormFlow whatsApp Settings, configure WhatsApp redirection, and enter your WhatsApp number. Once form submit it will redirected user to the whatsApp chat mobile app or whatsApp web. 

= How to active Connect Contact Form 7 as a floating widgets? =
Configure Contact Form 7 from CF7 Tabs of FormFlow and Enable Contact Forms 7 as floating widgets from General tabs. 

= How to active Connect Contact Form 7 as WhatsApp Form with WordPress? =
First configure Contact Form 7 from CF7 Tabs of FormFlow and Enable Contact Forms 7 as floating widgets from General tabs. Then Enable Contact Forms 7 as floating widgets and Select Contact Forms 7 forms for WhatsApp redirection from WhatsApp Tabs.

= Is there any action hook that can be used after the form submission event? =
Yes, 'formflow_form_submit' action hook added to use. It takes a parameter in which you can collect the form submission data by passing the following keys. E.g. $form['form_id']; $form['fields']; $form['time'] here 'fields' used to collect the form submission data.

== Installation ==

1. Upload files to the `/wp-content/plugins/simple-form` directory
2. Activate the plugin through the 'Plugins' screen in WordPress

=== How to Use ===
After activating the plugin navigate to your Admin Dashboard > FormFlow > Form and from there build your required form using drag and drop. Next save it with any form name.
You will then automatically be redirected to the "Data" page, from there you can edit your form or create a form on any of your pages or posts using the shortcode.
